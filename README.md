Fork自Kcptun_android，通过修改适配了goflyway，支持CDN。

[预编译安装包latest 下载](https://github.com/koolwiki/goflyway_android/releases/latest)

软件设置如下图，有的手机杀后台比较严重，可以把goflyway加入白名单或者不进行省电优化，这样可以避免在后台中被杀掉。

需要配合[Postern](https://play.google.com/store/apps/details?id=com.tunnelworkshop.postern&hl=zh)这个软件（可以去[这里下载](https://github.com/postern-overwal/postern-stuff/raw/master/Postern-3.1.2.apk)）一起使用。Postern代理服务器栏填127.0.0.1，端口栏填goflyway上面填写的本地端口号（纯端口号，如下图的8100）。
